# Compilers Practices Repository

## Description
This repository contains the practical implementations for the 2026-2 Compilers course, focused on the construction of a compiler in C. 
It explores the fundamental phases of compiler construction, including lexical and syntactic analysis.

## Technologies

- C
- Linux environment

## Branch Strategy

This project follows the GitHub Flow branch strategy.  
Feature branches are created directly from `main` for each practice and merged back after validation. The feature branch is then deleted.

For more information, see [GitHub Flow](https://docs.github.com/en/get-started/using-github/github-flow).

## Status

Under active development as part of academic coursework.

The repository structure and documentation will evolve as new compiler phases and technologies are explored throughout the course.
